Dashboard First

新版第一屏看这些

表盘先回答三个问题：结构是否已经脆弱、哪些信号灯亮了、价格是否已经跌破 HVL。

01

VIXEQ / VIX

判断单股期权恐慌和投机是否明显高于指数层面的恐慌。

当前快照：2.08x
高于约 1.6x 进入警戒。
02

COR1M

观察隐含相关性是否被压到极低，避免被“指数平静”误导。

当前快照：14.52
越低，越像分散押注会突然变成同向踩踏。
03

Call/Put

用 SMH 或半导体股票期权链聚合，看追涨杠杆是否拥挤。

当前代理：0.26
这是公开网页代理值，不是生产级实时源。
04

Left-tail Skew

看低 delta put IV 是否相对 call IV 抬升，捕捉左尾保险需求。

当前代理：9.55 IV点
快速升温时，说明市场开始买暴跌保护。
05

SPX / HVL

把 SPX 和 HVL 放在一起，看做市商 Gamma 风险是否进入负区间。

7383.74 / 7495
跌破 HVL 是执行层触发，不是普通支撑位。
06

Sentiment

情绪指标只做辅助背景，不单独作为交易触发。

当前快照：42/100
用来判断市场是过热、恐惧还是中性。

Method Map

这套框架的阅读顺序

1

先看结构

VIXEQ/VIX 高、COR1M 低，说明个股期权杠杆可能已经拥挤。

2

再看确认

Call/Put、Skew 和情绪指标用来判断投机是否过热、保护是否升温。

3

最后看执行

HVL/Gamma 翻转决定什么时候结构从“热”变成“容易踩踏”。
